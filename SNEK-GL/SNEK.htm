SNEK_GL

SNEK_GL - a C/C++ OpenGL and WinAPI SNEK project, that ... its a snake clone, do i need to explain?

Besides the usual snake game mechanic, it has Translation INI Packets (TIPs) and uses WinAPI for Pausing, About Dialogs, errors (Packet errors, etc.). SNEK_GL works on Windows NT 4.0 and higher, to Windows 11, and its minimal requirements are shown under:

CPU: Pentium Pro 180; Celeron 233 or any i686 (P6) CPU or better
RAM: 16 MB (32 MB recommended) or more
Graphics: any card compatible with OpenGL 1.1+ 2D or better
OS: Windows NT 4.0 or newer
